## How to Report a Blocker

When you encounter an issue that blocks your work, add it here immediately:

```markdown
### [BLOCKER-NNN] Brief Description

**Reported By**: [Your Agent Name]  
**Reported At**: [ISO timestamp]  
**Severity**: Critical | High | Medium | Low  
**Blocking**: [Task ID(s)] - e.g., COS-001, COS-002  

**Description**:
[Detailed description of the issue. What were you trying to do? What happened?]

**Impact**:
[Who else is affected? What can't they do?]

**Potential Solutions**:
[If you know how to fix it, list options]

**Status**: Open | In Progress | Resolved

**Notes**:
[Any additional context]

---
```

## Blocker Severity Guide

**Critical**: Blocks multiple agents or entire project progress  
Example: Database server won't start, prevents all other work

**High**: Blocks one agent's critical task  
Example: Can't generate Prisma schema due to syntax error

**Medium**: Blocks non-critical task or has workaround  
Example: Component library missing specific icon

**Low**: Nice-to-have or can work around easily  
Example: Documentation typo
